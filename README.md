 FinTrack App

A simple personal finance assistant web app built with HTML, CSS, and JavaScript.

Overview

FinTrack is a lightweight budgeting tool that helps users:

* Track income and expenses
* Smart income allocation based on your set budget
* Stay accountable 
* And monitor spending habits



This project started as a single HTML file and is now being structured into a scalable web application.

⸻

Features

* Add income and expenses
* View total balance
* Simple and clean UI
* Real-time updates

⸻

How to Run

1. Download or clone the repository
2. Open the project folder
3. Double-click index.html

The app will open in your browser.
